🏓 Day 22: Pong Game (OOP)

📌 Description  
A classic **Pong Game** built using Python's **Turtle Graphics** and **Object-Oriented Programming**.  
Two players control paddles and try to bounce the ball to score points.

🎮 Gameplay  
- 2-player game  
- Ball bounces between paddles  
- Miss = opponent gets a point  
- Speed increases over time ⚡  

✨ Features  
- Smooth ball movement  
- Paddle controls for both players  
- Collision detection (wall & paddle)  
- Scoreboard system  
- Ball speed increases after each hit  

🧠 Concepts Used  
- OOP (Classes & Objects)  
- Turtle Graphics  
- Game Loop (while loop)  
- Collision Detection  
- Keyboard Event Handling  
- Modular Code (multiple files)  

⚙️ Controls  
- Right Paddle → ⬆️ Up / ⬇️ Down  
- Left Paddle → W / S
