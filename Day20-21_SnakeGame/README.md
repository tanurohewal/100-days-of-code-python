🐍 Day 20-21: Snake Game (OOP)

📄 Project Overview  
This project is a **Snake Game** built using Python's **Turtle Graphics** and **Object-Oriented Programming (OOP)** as part of my **100 Days of Code – Python** journey.  
The player controls a snake to eat food, grow longer, and avoid collisions.

🎯 Objective  
- Apply Object-Oriented Programming in a real project  
- Build a complete game using multiple classes  
- Handle animations and user controls  
- Implement collision detection logic  

✨ Features  
- Snake movement using arrow keys  
- Food generation at random positions  
- Score tracking system  
- Snake grows after eating food  
- Game over on wall collision  
- Game over on self-collision  
- Smooth animation using Turtle  

🛠 Concepts Used  
- Classes & Objects  
- Multiple Python Files (Modular Code)  
- Turtle Graphics  
- While Loop (Game Loop)  
- Event Listeners (Keyboard Input)  
- Collision Detection  
- Random Module
