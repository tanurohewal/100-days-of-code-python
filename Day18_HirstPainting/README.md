🎨 Day 18: Hirst Painting Project

📄 Project Overview  
This project recreates a **Hirst-style dot painting** using Python's **Turtle graphics**.  
The colors used in the painting are extracted from an image using the **colorgram module**, and the program draws a grid of colorful dots randomly.

🎯 Objective  
- Practice using Turtle graphics in Python  
- Extract colors from an image using the colorgram library  
- Understand loops and coordinate positioning  
- Create a visual art project using Python  

✨ Features  
- Extracts RGB colors from an image using **colorgram**  
- Generates a palette of colors  
- Draws a **10 × 10 grid** of colorful dots  
- Uses random colors from the palette  
- Inspired by **Damien Hirst's dot paintings**

🛠 Concepts Used  
- Turtle Graphics  
- Colorgram Module  
- Random Module  
- Lists & Tuples  
- For Loops  
- RGB Color Mode
