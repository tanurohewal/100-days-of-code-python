🐢 Day 19: Turtle Racing Game

📌 Description  
This project is a **Turtle Racing Game** created using Python's **Turtle graphics module**.  
The user places a bet on a turtle color and watches the turtles race across the screen.  
The winner is decided randomly.

🚀 Features
- 6 turtles with different colors
- User can bet on a turtle
- Random turtle movement
- Animated race using Turtle graphics
- Displays if the user won or lost

🧠 Concepts Practiced
- Python Turtle Graphics
- Random Module
- Listss
- Loops
- User Input
- Basic Game Logic
