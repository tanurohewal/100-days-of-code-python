🧠 Day 17: Quiz Game using OOP

📄 Project Overview  
This project is a **Quiz Game** built using **Object-Oriented Programming (OOP)** in Python as part of my **100 Days of Code – Python** journey.  
The program asks True/False questions, checks the user's answers, and keeps track of the score.

🎯 Objective  
- Understand Object-Oriented Programming concepts  
- Learn how to create and use classes  
- Organize code using multiple Python files  
- Build an interactive command-line quiz application  

✨ Features  
- Multiple True/False questions  
- Score tracking system  
- Instant feedback (Correct / Wrong)  
- Displays the correct answer  
- Shows final score at the end of the quiz  

🛠 Concepts Used  
- Classes and Objects  
- Constructors (`__init__`)  
- Lists and Dictionaries  
- Loops  
- If-Else Conditions  
- Modular Programming
