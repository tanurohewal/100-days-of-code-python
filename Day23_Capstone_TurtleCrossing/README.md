🐢 Day 23: Turtle Crossing Game (Capstone Project)

📌 Description  
A fun **arcade-style game** inspired by *Frogger*, built using Python's **Turtle Graphics** and **Object-Oriented Programming**.  
The player controls a turtle that must cross the road while avoiding moving cars.

🎮 Gameplay  
- Move the turtle forward to cross the road  
- Avoid fast-moving cars 🚗  
- Each successful crossing increases the level  
- Game ends if the turtle gets hit  

✨ Features  
- Player movement using keyboard (Up key)  
- Random car generation  
- Increasing difficulty (speed increases each level)  
- Collision detection with cars  
- Level tracking system  
- Game over screen  

🧠 Concepts Used  
- OOP (Classes & Objects)  
- Turtle Graphics  
- Game Loop  
- Collision Detection  
- Random Module  
- Event Listeners (Keyboard Input)  
- Modular Programming  

⚙️ Controls  
- ⬆️ Up Arrow → Move Forward
